# ⌨️ Minimal Typing Test

A sleek, responsive, and performance-focused typing speed tester. Built with vanilla JavaScript, CSS Flexbox, and a dictionary of 10,000 common English words.

## ✨ Features

* **10,000 Word Dictionary:** Randomly generates tests using a massive list of common English words.
* **Dynamic Word Counting:** Choose between 25, 50, or 100-word tests.
* **Real-time WPM:** Track your Words Per Minute as you type.
* **Smart Word Wrapping:** Custom "Word Container" logic prevents words from being cut off mid-letter at the edge of the screen.
* **Blinking Caret:** A smooth, reactive caret that follows your exact position.
* **Theme Switcher:** Includes **Carbon (Dark)**, **Cyberpunk**, and **Paper (Light)** themes with local storage persistence.
* **Results Modal:** Instant feedback on your WPM and Accuracy upon completion.
